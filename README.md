# Coloring Page Maker

An interactive coloring page maker with rainbow stamps and PDF export functionality.

## Live Demo

View the live application at: https://mwudka.github.io/coloring-page-maker/

## Features

- 12 colorful stamps plus a remove tool for easy editing
- Image-based stamps with pixel-perfect collision detection
- Rainbow color progression for non-overlapping stamps
- Randomized stamp size variation
- Export to PDF with high DPI rendering
- Stream Deck integration with downloadable profile

## Stream Deck Support

Download the Ulanzi Stream Deck profile directly from the application or from the [live demo](https://mwudka.github.io/coloring-page-maker/). The profile includes:

- 12 stamp buttons (Ctrl+Alt+Shift+A through L)
- 1 remove tool button (Ctrl+Alt+Shift+M)
- Custom icons for each stamp

The profile is compatible with Ulanzi Stream Deck devices and can be imported into Ulanzi Studio.
